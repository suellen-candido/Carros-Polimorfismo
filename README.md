# Carros com Herança, Polimorfismo e Encapsulamento

## O que faz?


Esse programa simula diferentes tipos de carros e permite testar suas ações básicas.
Ele cria três veículos: um carro inteligente, um carro esportivo e um carro de corrida. Logo depois, executa testes com cada um.

Durante a execução, o programa:

- Aumenta a velocidade dos carros usando seus métodos específicos.

- Mostra a velocidade atual após cada ação.

- Ativa funções especiais de cada tipo de carro (como turbo e aceleração de corrida).

- Exibe mensagens indicando o comportamento realizado por cada veículo.

O objetivo é demonstrar como diferentes carros podem reagir de formas distintas quando são acelerados ou quando utilizam suas funções próprias. Aplicando conceitos de Herança, Polimorfismo e Encapsulamento
